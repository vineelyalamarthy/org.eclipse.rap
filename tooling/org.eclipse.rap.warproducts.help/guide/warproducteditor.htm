WAR Product Configuration Editor

Building an WAR product using PDE is centered around a WAR product configuration, a .warproduct file. This file manages all aspects of a WAR product definition from its constituent plug-ins to configuration files.

The WAR product configuration is entirely managed by PDE to simplify WAR the war deployment process. Its format is internal and is not intended to be read by tools other than PDE.

PDE provides a form-based multi-page editor to manage the WAR product configuration.
